Backend приложения отслеживания цен карточки товара с сайта wildberries

Возможности проекта:
Автоматическое отслеживание динамики цен товаров с сайта wildberries.ru c заданным диапазоном дат и временным интервалом (1 час, 12 часов, 24 часа). Задачи коллекционирования данных реализованы в виде ассинхронных celery задач.

1. Регистрация пользователя (логин, пароль, почта)
2. Добавление, редактирование, удаление товаров для отслеживания динамики цен (наименование и артикул)
3. Постановка пользователем задачи отслеживания (выбирается товар, усанавливается дата окончания отслеживания, выбирается интервал отслеживания. Редактирвание задачи, удаление задачи.
4. Просмотр результатов отслеживания карточки товара пользователя(наименование, брэнд, артикул, цена товара, цена товара со скидкой). Фильтрация товаров по наименованию и датам. Удаление результатов.
5. Тесты (Unit tests)

Стэк:
- djangorestframework,
- celery,
- django-celery-beat,
- postgresql,
- docker, docker-compose,
- requests,
- lxml


Инструкции запуска приложения

1. Скачать проект с github
2. В папке проекта выполнить команду "docker-compose up" и дождаться окончания сборки
3. Открыть приложение в браузере 0.0.0.0:8000/api/
4. Зарегистрировать нового пользователя 0.0.0.0:8000/api/singup/ введя логин, почту и пароль
5. Авторизовать нового пользователя.
6. Добавить товар для отслеживания цен 0.0.0.0:8000/api/goods_for_tracking/
Нужно ввести название товара и его артикул с сайта wildberries.ru, например: наименование: Зарядное устройство BCI-4D, артикул: 63208991
7. Добавть задачу 0.0.0.0:8000/api/my_tasks/
Выбрать из списка название товара для отслеживания цен, выбрать из выпадающего списка период отслеживания: 1 час, 12 часов, 24 часа. Для теста выберите 30 сек test. Выбрать дату окончания выполнения задачи.
8. После запуска задачи в зависимости от времени отслеживания данные о товаре спарсенные с сайта wildberries.ru (название товара, брэнд, артикул, цена, цена со скидкой) будет отображаться в 0.0.0.0:8000/api/my_tracked_goods.
При добавлении разных товаров и задач можно фильтровать выборку отслеживаемых товаров по датам и их названию.
